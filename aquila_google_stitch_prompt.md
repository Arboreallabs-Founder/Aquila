# Google Stitch Prompt: Aquila Landing Page

Create a premium, modern, responsive landing page for **Aquila**, a workforce transformation and capability-building organization in India.

Aquila operates across three major ecosystems:

1. **Corporates**
2. **Educational Institutions**
3. **Social Impact Ecosystems**

The website should position Aquila as:

**“A modern capability infrastructure platform for India.”**

The landing page should feel strategic, premium, credible, institutional, and future-facing. It should not feel like a generic training company website. It should feel like a serious capability-building platform working across enterprises, universities, schools, NGOs, government ecosystems, and social impact programs.

---

## Brand Direction

Use a refined, premium visual identity.

### Typography

Use:

- **Headline font:** Playfair Display
- **Body / subheading font:** Public Sans or Canva Sans-style modern sans-serif

The overall type hierarchy should feel elegant, editorial, and institutional.

### Color System

Use a sophisticated palette based on the Aquila brand system:

- Deep teal / blue-green as the primary trust color
- Purple as a secondary brand color
- Electric violet / accent purple for highlights and CTAs
- Ivory / warm off-white as the main page background
- Space blue / navy as the dark contrast color

Suggested palette:

- Deep Teal: `#386982`
- Purple: `#9B5CB8`
- Accent Violet: `#7B45E6`
- Ivory: `#F8F5EC`
- Space Blue: `#243449`
- Soft Lavender: `#EFE7FF`
- Muted Slate: `#6B7280`
- White: `#FFFFFF`

Use gradients subtly, especially between deep teal, purple, and accent violet.

---

## Overall Page Style

Create a clean, premium landing page with:

- Large editorial hero section
- Smooth rounded cards
- Modern abstract background shapes
- Soft gradients
- Capability infrastructure theme
- Indian workforce transformation narrative
- High-quality image placeholders
- Sophisticated iconography
- Responsive desktop and mobile design
- Clear CTA hierarchy
- Sections that flow like a strategic pitch deck

Avoid clutter. Use generous whitespace. Make it feel polished enough for corporates, universities, impact partners, and government ecosystem stakeholders.

## Design Quality Expectations

The website should feel like a mix of:

- A premium consulting firm website
- A workforce transformation platform
- A modern education and employability organization
- A social impact capability-building ecosystem

It should not look like a basic coaching institute, NGO landing page, or corporate training template.

Use strong storytelling, elegant spacing, large headlines, and premium visual hierarchy.
